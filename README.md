# CIVI-GENESIS 🏙️

**AI-Powered Policy Simulator (LLM + Neural Network + Simulation)**

## 📌 Project Overview

CIVI-GENESIS is an AI-based policy simulation platform that models how thousands of citizens might react to new government or business policies.
The system generates synthetic citizens with different demographic and behavioral attributes and predicts their reactions using a hybrid approach combining **Large Language Models (LLM), Machine Learning, and rule-based simulation**.

This project demonstrates concepts from **Artificial Intelligence, Machine Learning, Data Structures & Algorithms, and Networking**.

---

## 🎯 Key Features

* Generate up to **5000 synthetic citizens** with diverse attributes
* Simulate reactions to different **public policies**
* AI reasoning using **Google Gemini API**
* Custom **Neural Network implemented in JavaScript**
* Rule-based fallback system if AI is unavailable
* Interactive **data visualization dashboards**
* Citizen-level and group-level analytics

---

## 🧠 AI & Machine Learning

The system uses a hybrid AI pipeline:

1. **LLM Reasoning (Gemini)**
   Generates realistic citizen reactions and diary entries.

2. **Neural Network Training**
   A Multi-Layer Perceptron (MLP) learns from LLM outputs and predicts reactions for the entire population.

3. **Rule-Based Simulation**
   Ensures the simulation works even if the API is unavailable.

Neural Network Architecture:

Input Layer: 21 Features
Hidden Layer 1: 64 Neurons
Hidden Layer 2: 32 Neurons
Output Layer: 3 Predictions

Outputs:

* Happiness Change
* Policy Support
* Income Change
